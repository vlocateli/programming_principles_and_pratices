# Programming_principles_and_pratice

### Solutions to exercises

To run:

## Linux

```console
$ gcc <file_name>.cc -o <program_name>
$ ./<program_name>
``` 

## Windows (with WSL)
```console
$ gcc <file_name>.cc -o <program_name>
$ ./<program_name>
```

Some folders have makefiles,to compile it run:

```console
$ make 
	```
Note: Makefile only runs on Linux.
